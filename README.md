# ارسال پیامک OTP با پایتون - آرتا پیامک

این ریپازیتوری شامل یک نمونه کد ساده و بهینه برای ارسال **پیامک اعتبارسنجی (OTP)** با استفاده از **[وب‌سرویس پیامکی آرتا پیامک](https://artapayamak.com)** است. ما از API قدرتمند IPPanel Edge استفاده کرده‌ایم تا پیامک‌های شما با بالاترین سرعت و ضریب تحویل (Delivery Rate) ارسال شوند.

## چرا آرتا پیامک؟
[آرتا پیامک](https://artapayamak.com) ارائه دهنده تخصصی وب‌سرویس‌های پیامکی برای کسب‌وکارهای مدرن است. با استفاده از APIهای ما، شما به ابزارهای زیر دسترسی خواهید داشت:
*   ارسال پیامک‌های خدماتی (Pattern) برای OTP
*   گزارش‌گیری دقیق وضعیت ارسال (Delivery Status)
*   پنل مدیریت کاربران و دفترچه تلفن پیشرفته
*   پشتیبانی از پروتکل‌های REST و SOAP

## نحوه استفاده و اجرای کد

1. ابتدا کد را دریافت کنید.
2. توکن API خود را از پنل کاربری در [آرتا پیامک](https://artapayamak.com) دریافت کنید.
3. متغیر `IPPANEL_API_TOKEN` را در فایل `.env` یا متغیرهای محیطی سیستم خود قرار دهید.

### نمونه کد (`send_otp.py`)

```python
import os
import requests

# آدرس endpoint وب‌سرویس IPPanel
url = "https://edge.ippanel.com/v1/api/send"

# فراخوانی توکن امنیتی از متغیرهای محیطی
api_token = os.environ.get("IPPANEL_API_TOKEN")

# تنظیمات پیامک (الگوی ارسالی)
payload = {
    "sending_type": "pattern",
    "from_number": "+983000505",
    "code": "xxxxxxxxxxxxxxx", # شناسه الگوی ثبت شده در پنل
    "recipients": ["+989120000000"],
    "params": {
        "code": "458921" # کد OTP تولید شده
    }
}

headers = {
    "Authorization": api_token,
    "Content-Type": "application/json"
}

# ارسال درخواست به وب‌سرویس
response = requests.post(url, json=payload, headers=headers)

print(response.json())
```

## توضیح خط‌به‌خط کد
برای درک بهتر نحوه کارکرد این کد، بخش‌های کلیدی آن را بررسی می‌کنیم:

*   **`url`**: آدرس مقصد API است که درخواست‌های ما به آن ارسال می‌شود.
*   **`api_token`**: برای امنیت بیشتر، توکن شما از متغیر محیطی (Environment Variable) خوانده می‌شود تا در سورس‌کد لو نرود.
*   **`sending_type: "pattern"`**: ما از سیستم «الگو» استفاده می‌کنیم که بهترین روش برای ارسال پیامک‌های خدماتی و OTP است (چون معمولاً سریع‌تر از خطوط تبلیغاتی ارسال می‌شود).
*   **`code`**: در اینجا باید "شناسه الگوی" (Pattern ID) که در پنل آرتا پیامک تعریف کرده‌اید را قرار دهید.
*   **`params`**: این بخش جادوی اصلی است. هر متغیری که در متن الگوی خود در پنل تعریف کرده‌اید (مثلاً `%code%`)، باید در اینجا مقداردهی شود.
*   **`headers`**: احراز هویت (Authentication) شما در این بخش انجام می‌شود که `api_token` شما را به وب‌سرویس معرفی می‌کند.
